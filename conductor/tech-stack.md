# Tech Stack

## Frontend
- **Framework:** React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS + shadcn/ui
- **State Management:** Zustand
- **Terminal Emulation:** xterm.js
- **Build Tool:** Vite

## Backend
- **Framework:** Axum (Rust)
- **Runtime:** tokio
- **Git Operations:** git2-rs
- **Terminal PTY:** portable-pty

## Development & Tooling
- **Package Manager:** Bun
- **Testing (Frontend):** Vitest
- **Build Tool (Backend):** Cargo
